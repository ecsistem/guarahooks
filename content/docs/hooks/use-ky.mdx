## Props

| Prop       | Type      | Default | Description                                  |
| ---------- | --------- | ------- | -------------------------------------------- |
| `options?` | `Options` | `{}`    | Options for creating the ky client instance. |

See [ky documentation](https://github.com/sindresorhus/ky#options) for more details about the options.

## Returns

A ky instance with the following methods:

| Method         | Description                                    |
| -------------- | ---------------------------------------------- |
| `get`          | Perform a GET request and parse JSON response. |
| `post`         | Perform a POST request with optional body.     |
| `put`          | Perform a PUT request with optional body.      |
| `delete`       | Perform a DELETE request.                      |
| `patch`        | Perform a PATCH request with optional body.    |
| `extend`       | Create a new ky instance with merged defaults. |
| `HTTPError`    | Class for HTTP error handling.                 |
| `TimeoutError` | Class for timeout error handling.              |

## Examples

### Basic GET Request

```tsx
const kyClient = useKy({ prefixUrl: 'https://api.example.com' });
const data = await kyClient.get('items/1').json<{ id: number; name: string }>();
```

### POST Request

```tsx
const kyClient = useKy({ prefixUrl: 'https://api.example.com' });
const response = await kyClient
  .post('items', { json: { name: 'New Item' } })
  .json();
```
